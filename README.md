A Pong-variant space combat game. Two ships, bullets, a color-shifting shield orb, shield lock mechanics, and scoring to 10. Built with vanilla JavaScript on HTML5 Canvas.

## Features

- Two-player local multiplayer
- Ship bullets with firing cooldowns
- Color-shifting shield orb — lock it to your color for protection
- First to 10 wins
- Custom Codex font
- No dependencies — pure HTML/CSS/JS

## Play

Open `index.html` in any browser, or play at [anarchysc.github.io/in-the-black-game-anarchygames](https://anarchysc.github.io/in-the-black-game-anarchygames/).
